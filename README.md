# Angular2 + Webpack starter

## Install
```
> npm install
```

## Developers mode
```
> npm start
```
Will be start server on [localhost:8080](http://localhost:8080/)

## Production build
```
> npm run build
```
Files files are build at directory /dist/

## Tests
```
> npm test
```
